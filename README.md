# Loan Default Prediction

## Overview
This project uses Machine Learning to predict whether a borrower will default on a loan.  
It helps analyze credit risk based on financial and personal features.

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- FastAPI (API)
- Streamlit (UI)

## Workflow
1. Data Cleaning & Preprocessing  
2. Exploratory Data Analysis  
3. Model Training (Random Forest - Final Model)  
4. Model Evaluation  

## Model Performance
Random Forest Accuracy: **98%**
